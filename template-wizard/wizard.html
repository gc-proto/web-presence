Find the right template for the job

Answer the following questions to find out if there is a template you should use.

What type of content do you want to publish?

  • Information about programs and services
  • Corporate content (information related to the organization or the minister)
  • Navigation pages
  • Promotional content
  • Consulting Canadians

What will be the the purpose of the page?

  • Launch a service (like applying for a program)
  • Give background information about a program
  • List elements (like a list of documents or publications)
  • Give information about an Act or Regulation
  • None of the above

What type of list of elements do you want to publish?

  • Short list (10 to 50 items)
  • Long list (100+ items)
  • Long list that needs to be filtered
  • Long list with faceted filtering
  • GC-wide collection
  • List of guidance on legislation

Do you want to give information about an Act or a Regulation?Â 

  • Act
  • Regulation

What type of information about your organization do you want to publish?

  • Official institutional presence (home page for your organization)
  • Contact information for your organization
  • Information about the Minister
  • Report on performance on service standards
  • None of the above

Are you a government organization or institution, or a partnering/collaborative arrangement?

  • Partnering or collaborative arrangement
  • Government institution listed in Schedule I, I.1 or II
  • An organization affiliated with a government institution
  • AI don't know

What kind of navigation page do you want to publish?

  • Topic page, part of the global topic tree
  • List of elements (like a list of publications or documents)
  • None of the above

What do you want to promote?

  • Initiative or program
  • Event

Promotional activities

To promote an initiative or an event, use one of the following templates. Note that these are intended for temporary use (for the duration of the promotional campaign).

  • Campaign page
  • Promotional events page

Consultations

Use the consultation template.

Service initiation

Use the service initiation template.

Program description

Use the program description page.

Performance reporting

Use the Institutional service performance reporting page.

Topics

Use the topic page.

Topic page should not only be used if they are part of the topic tree. They are navigational in nature, and should not be used when several tasks should be grouped together. Always consult your Theme Lead before creating a new topic page.

Finders

Use one of the following templates:

  • Short index: use for short lists (10 to 50 items)
  • Long index: use for longer lists (50+ items)
  • Finder page: use for long lists that can be filtered
  • Faceted finder page: use for long lists, with faceted filtering
  • Collection landing page: for official GC collections
  • Guidance on legislation profile pages: gives access to guidance on legislation

Acts and regulations

Use one of the following templates:

  • Act profile page
  • Regulation profile page

Ministers

Use the Ministerial profile template.

Contact information

Use one of the Contact us pages.
