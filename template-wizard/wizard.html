  • Promotional events page

Performance reporting

Use the Institutional service performance reporting page.

Topics

Use the topic page.

Topic page should not only be used if they are part of the topic tree. They are navigational in nature, and should not be used when several tasks should be grouped together. Always consult your Theme Lead before creating a new topic page.

Finders

Use one of the following templates:

  • Short index: use for short lists (10 to 50 items)
  • Long index: use for longer lists (50+ items)
  • Finder page: use for long lists that can be filtered
  • Faceted finder page: use for long lists, with faceted filtering
  • Collection landing page: for official GC collections
  • Guidance on legislation profile pages: gives access to guidance on legislation

Acts and regulations

  • Act profile page
  • Regulation profile page
